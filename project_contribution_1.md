# مساهمات العضو الأول في مشروع DiagnoLink

## 1. إعداد وتدريب نموذج الذكاء الاصطناعي
- جمع وتحضير بيانات الأشعة السينية (NIH Chest X-ray Dataset).
- معالجة البيانات وتنظيفها وتوزيعها على مجموعات التدريب والاختبار.
- بناء نموذج ConvNeXt Large باستخدام PyTorch.
- ضبط معمارية النموذج لتشمل رأس تصنيف ورأس تحديد مواقع (Bounding Box Localization).
- تدريب النموذج على السيرفرات السحابية، وضبط معاملات التعلم (Hyperparameters).
- تقييم النموذج واستخراج أفضل نقطة تحقق (Best Checkpoint) بناءً على أعلى AUC.

## 2. تطوير الخادم الخلفي (Backend)
- بناء خادم FastAPI لاستقبال الصور وبيانات المرضى.
- تطوير دوال معالجة الصور وتحويلها إلى تنسيقات مناسبة للنموذج.
- ربط النموذج المدرب بالخادم وتوفير نقطة نهاية `/predict`.
- إضافة معالجة النتائج وتنسيقها (probabilities, bounding boxes).
- إعداد نقطة نهاية `/healthcheck` لمراقبة حالة الخادم.
- كتابة اختبارات للتأكد من سلامة عمل الخادم.

## 3. التكامل مع قاعدة البيانات
- تصميم جداول المرضى والتحليلات في Supabase (PostgreSQL).
- ربط الخادم الخلفي بقاعدة البيانات لتخزين واسترجاع بيانات المرضى والتحليلات.
- إعداد صلاحيات الوصول (Role-based Access) وتأمين البيانات.

## 4. التوثيق والتقارير
- كتابة توثيق مفصل لكود الخادم الخلفي.
- إعداد شروحات حول كيفية تدريب النموذج وتشغيل الخادم.
- المساهمة في كتابة فصل الذكاء الاصطناعي وفصل البنية الخلفية في التقرير النهائي.
